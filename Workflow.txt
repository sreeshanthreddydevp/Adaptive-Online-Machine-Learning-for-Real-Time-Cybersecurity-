PHASE 1: DATA COLLECTION
==================================================
1. Load dataset .
2. Explore dataset shape, features, and label distribution.

   - Target class distribution (Bar chart)

PHASE 2: DATA PREPROCESSING
==================================================
1. Replace infinite values with NaN.
2. Remove null values.
3. Remove duplicate records.
4. Encode categorical columns using LabelEncoder.
5. Separate:
      X = Features
      y = Label
6. Feature selection:
      - Train RandomForest
      - Extract Top 20 important features
7. Feature scaling:
      - Apply StandardScaler
8. Handle class imbalance:
      - Apply SMOTE oversampling
9. Split dataset:
      - Train (70%)
      - Test (30%)


PHASE 3: OFFLINE MODEL TRAINING (Batch Learning)
==================================================
Train and evaluate:

1. Gaussian Naive Bayes
2. Logistic Regression
3. Decision Tree
4. Random Forest
5. XGBoost
6. Deep Neural Network (DNN)
7. RNN (LSTM)
8. CNN 

For each model:
    - Train model
    - Predict test data
    - Calculate: Accuracy, Precision, Recall, F1-Score, ROC-AUC
    - Plot: Confusion Matrix, Multiclass ROC Curve

Store all results in comparison table.

Best Offline Model:
    → XGBoost

Save artifacts:
    - offline_model.sav
    - offline_scaler.sav
